JavaScript was invented by Brendan Eich in 1995.

It was developed for Netscape 2, and became the ECMA-262 standard in 1997.

After Netscape handed JavaScript over to ECMA(European Computer Manufacturers Association), 
the Mozilla foundation continued to develop JavaScript for the Firefox browser. 
Mozilla's latest version was 1.8.5. (Identical to ES5).

1997	ES1	ECMAScript 1 was released

2018	ES6	Full support for ES6 in all browsers

Latest ES13 introduced in 2022 which is supported for all browser
